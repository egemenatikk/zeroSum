# zeroSum

Given an integer, this program tries to find if 0 can be reached by adding "+"'s and "-"'s between the digits of this integer and executing the operation.

**Input:** an integer  
**Output:** a string consists of "+"'s and "-"'s or "not possible" 


Example cases:

* **Input:** 123  
**Output:** --+

* **Input:** 11  
**Output:** -+

* **Input:** 1234  
**Output:** -++-

* **Input:** 87  
**Output:** not possible

* **Input:** 1231564  
**Output:** ----++-
